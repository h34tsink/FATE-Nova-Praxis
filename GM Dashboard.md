# GM Dashboard

**Summary:** Single-pane command center for running Nova Praxis sessions. Scan this note for at-a-glance state, then use Claude Code commands for interactive queries.

---

## Claude Code Commands

| Command | What It Does | Example |
|---------|-------------|---------|
| `/gm-start` | Bootstrap a session — loads all context | `/gm-start 9` |
| `/rules` | Rules oracle — instant ruling with source | `/rules can a SIM invoke aspects?` |
| `/npc` | NPC dialogue — voice-accurate lines | `/npc kestrel "What's your price?"` |
| `/scene` | Scene framing — zones, aspects, NPCs | `/scene docking bay confrontation` |
| `/recap` | Session state — where are we now? | `/recap` |

Start every session with `/gm-start`. Then use `/rules`, `/npc`, `/scene`, `/recap` as needed during play.
