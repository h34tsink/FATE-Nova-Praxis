The numeric and adjective scale used throughout **Strands of Fate** (and Nova Praxis) to rate [[Glossary/Skill|skills]], [[Glossary/Difficulty|difficulties]], [[Glossary/Effort|effort]], and the outcomes of tests. Every roll, every opposed action, and every static challenge ultimately resolves to a position on the Ladder.

## The Ladder

| Value | Adjective |
|---|---|
| +8 | Legendary *(extension)* |
| +7 | Epic *(extension)* |
| +6 | Super Human |
| +5 | Superb |
| +4 | Great |
| +3 | Good |
| +2 | Fair |
| +1 | Average |
| 0 | Mediocre |
| -1 | Poor *(extension)* |
| -2 | Terrible *(extension)* |

Nova Praxis's core scale runs **Mediocre (0)** to **Super Human (+6)**; *Epic* and *Legendary* are reserved for genuinely setting-breaking feats and adversaries. The negative rungs (*Poor*, *Terrible*) are standard FATE extensions used for failed defenses, broken gear, and severe state penalties.

## How It Is Used

- **Skill ranks** are written as ladder values (e.g., a character with Firearms +4 is *Great* with a gun)
- **Difficulties** are set on the same scale, so the GM can say "this is a *Fair* lock" instead of picking a number
- **Effort** generated by a roll is compared against the difficulty; the difference is measured in [[Glossary/Shift|shifts]]
- **Final outcomes** are reported back as ladder steps — *"You succeed with Good Effort,"* *"That's a Superb Stealth roll"*

Mirrored in code at `Data/fate-ladder.ts` for the character creator and any tooling that needs a single source of truth.
