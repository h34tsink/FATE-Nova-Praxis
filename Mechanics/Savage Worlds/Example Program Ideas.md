# Savage Worlds TTRPG Nova Praxis Programs

Below are various programs available in the Nova Praxis setting of the Savage Worlds TTRPG. Each program provides a bonus Benny when a successful roll is achieved, aiding in specific knowledge or skill areas.

## Hacking Assistance Algorithm (Knowledge, Hacking)
- **Description**: Aids in bypassing security systems or breaking into secure networks.
- **Benefit**: Provides a bonus Benny for Hacking rolls to overcome a specific security system or network.

## Augmented Reality Stealth Evasion Planner (Knowledge, Stealth)
- **Description**: Assists in plotting the most discreet route through an area.
- **Benefit**: Provides a bonus Benny for Stealth rolls to navigate through a specified area.

## Advanced Medical Diagnosis Assistant (Knowledge, Medicine)
- **Description**: Provides a detailed diagnosis and treatment plan for a patient.
- **Benefit**: Provides a bonus Benny for Medicine rolls for treating the specified condition.

## Combat Simulation and Tactics Analyzer (Knowledge, Tactics)
- **Description**: Uses data to run combat simulations for devising battle strategies.
- **Benefit**: Provides a bonus Benny for Tactics rolls for a specific combat encounter.

## Advanced Drone Control Interface (Knowledge, Piloting)
- **Description**: Optimizes drone control and navigation.
- **Benefit**: Provides a bonus Benny for Piloting rolls for controlling a specific drone model.

## Language Decryption and Translation Tool (Knowledge, Linguistics)
- **Description**: Decrypts and translates unfamiliar languages or codes.
- **Benefit**: Provides a bonus Benny for Linguistics rolls for deciphering a specific language or code.

## Financial Market Prediction Model (Knowledge, Finance)
- **Description**: Analyzes financial data to predict market trends.
- **Benefit**: Provides a bonus Benny for Finance rolls for a specific investment decision.

## Biometric Spoofing System (Knowledge, Disguise)
- **Description**: Generates synthetic biometric data for impersonation.
- **Benefit**: Provides a bonus Benny for Disguise rolls for fooling biometric security systems.

## Adaptive Environmental Hazard Analyzer (Knowledge, Survival)
- **Description**: Develops strategies for surviving hostile environments.
- **Benefit**: Provides a bonus Benny for Survival rolls for traversing a specific hazardous environment.

## Social Engineering Intrusion Planner (Knowledge, Persuasion)
- **Description**: Creates psychological profiles for social manipulation.
- **Benefit**: Provides a bonus Benny for Persuasion rolls for infiltrating a specific target.

## Quantum Encryption Breaker (Knowledge, Cryptography)
- **Description**: Cracks quantum encryption systems.
- **Benefit**: Provides a bonus Benny for Cryptography rolls for decrypting quantum-encrypted data.

## Advanced AI Behavior Prediction Engine (Knowledge, Artificial Intelligence)
- **Description**: Predicts AI system behaviors and weaknesses.
- **Benefit**: Provides a bonus Benny for Artificial Intelligence rolls for interacting with specific AI systems.

## Nanoscale Matter Manipulation Protocol (Knowledge, Nanotechnology)
- **Description**: Manipulates matter at a molecular level using nanotechnology.
- **Benefit**: Provides a bonus Benny for Nanotechnology rolls for creating nanotech-based items.

## Zero-G Combat Maneuver Simulator (Knowledge, Zero-G Combat)
- **Description**: Develops combat strategies in zero-gravity environments.
- **Benefit**: Provides a bonus Benny for Zero-G Combat rolls for specific zero-gravity combat scenarios.
