# Shadow Relay Node

**Zone Type:** Digital infrastructure — comms relay, data scraping, network surveillance

## Aspects

- *Old Signals, New Enemies*
- *Someone Else Is Already Scraping These Logs*

## Description

A repurposed comms relay buried in a disused maintenance corridor. The hardware is decades old but still active — patched, re-routed, and tapped by at least three different parties. The node handles local mesh traffic, burst transmissions, and dead-drop data caches. It smells like ozone and dust. The kind of place where the walls have ears because someone literally wired them that way.

## Clue

A burst transmission from Valare is cached here: *"I won't merge. Don't make me."* Timestamped, partially routed. Available through SINC / Investigate (digital) rolls.

## Complication

The node logs are already being scraped — someone else is here. A rival faction (Talons, [[Factions/The Coalition/_Coalition Overview|Coalition]], or [[Locations/Shadow Market]]) is running parallel searches. Accessing the node risks detection from both the rival team and Valare's savant programs. Tick Exposure on sloppy access.

## NPC Introduction — Rival Faction Operative

No named NPC here — this is an **environmental reveal**. The rival isn't a person the PCs meet; it's evidence that someone else is already working the same angle.

> **Option A — The Warm Console:** *"You reach the relay node and start pulling logs. The interface is old, slow, cycling through cached transmissions. But the search filters are already set. Someone was here — recently — running the same queries you're about to run. The filters are still warm. The chair is still warm."*
> The rival's absence is the introduction. The PCs are behind, and they know it.

> **Option B — The Live Encounter:** If you want a face-to-face: *"You round the corridor to the relay alcove and someone is already there — crouched over the console, data cable jacked into a wrist port, working fast. They hear you and don't turn around. 'I'm almost done,' they say. 'You can have what's left.'"*
> The operative doesn't care about the PCs. They care about the data. This creates a choice: wait (lose time), interrupt (risk confrontation), or try to read over their shoulder (moderate difficulty).

> **Option C — The Digital Ghost:** For Grift specifically: *"You jack in and start scraping the logs. The data flows clean for about four seconds. Then you hit something — a partition, a filter, someone else's active session running alongside yours. They're not blocking you. They're watching what you search for."*
> The rival is present digitally, not physically. Grift has to decide: keep searching (and reveal his targets to the watcher), try to trace them (hard difficulty), or disconnect and lose the lead.

**Why it works:** The relay node doesn't have a social NPC to introduce — the *evidence of a competitor* is the pressure. It tells the PCs they're not alone, they're not first, and someone else has resources they might not.

- *Old Signals, New Enemies* → Pulling the data thread triggers a cascade of flags across the local mesh
- *Curiosity Kills More Than Cats* (Grift) → The data thread is *right there* — but pulling it alerts everyone watching
