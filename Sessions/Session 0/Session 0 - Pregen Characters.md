# Session 0: Pregen Characters

**Pick one.** These are loaners for the teaching scenario. You'll build your real character after. Choose whoever sounds fun -- don't overthink it.

Each character has a **role tag** so you know what they're good at, and a **"Play this if you like..."** line to help you pick fast.
